STUDENT SOFTWARE DEVELOPER @ SCALDA

Mijn naam is Jeroen Suurmond, een 18-jarige creatieve probleemoplosser uit Goes met een passie voor ruimtevaart, luchtindustrie, gaming en auto's. Of ik nu een game speel of een softwareprobleem oplos, ik rust niet voordat ik de oplossing vind. Daarom heb ik gekozen voor een carrière als softwareontwikkelaar. Mijn droom is om iets toe te kunnen voegen aan een (of meerdere) van deze sectoren, maar ik zou het ook leuk vinden om een andere sector te ontdekken.

Wie ben ik?

Mijn achtergrond

Ik ben Jeroen Suurmond, ik ben 18 jaar en ben geboren op 5 april 2005 in de gemeente Dordrecht. Ik heb hier 15 jaar gewoond voordat ik verhuisd ben naar Goes. Ik heb hierdoor mijn middelbare school eerst coor 2 jaar op het Walburg College in Zwijndrecht gezeten, en de andere 2 jaar op het Goese Lyceum in Goes, Ookwel bekend als Pontes Oranjeweg. Ik heb altijd al interesse gehad richting ICT dankzij mijn vader die al zo’n 25 jaar in die sector werkt, en dankzij mijn goede probleem-oplossings skills. Dit heeft heeft geleid tot mijn keuze om een opleiding Software Development te doen.

Projecten

pokemon game

Voor dit project moesten wij een pokemon spel maken in java dat je kan spelen vanuit de command line, dit project kan je vinden op github. Dit project deden wij aan de hand van sequence diagrams en userstories.

portfolio website

Bij dit project moesten wij de portfolio website bouwen waar je nu naar kijkt! Deze portfoliowebsite moesten we binnen 1 sprint maken. Ik heb deze website gemaakt met puur html, css en een beetje javascript. Geen externe imports! De sourcecode van mijn website kan je vinden op github.
